# BookMaster - Court Booking System

A modern web application for managing racquet court bookings with WhatsApp export functionality.

## Features

- **Court Booking Management**: Create, edit, and delete bookings
- **Multiple Sports**: Support for Padel, Tennis, and Badminton
- **Player Management**: Track players and calculate spots needed
- **WhatsApp Export**: Generate formatted text for WhatsApp sharing
- **Responsive Design**: Works on desktop and mobile devices
- **Modern UI**: Clean, professional interface with date indicators

## Quick Start

### Option 1: Local Deployment
1. Download all files (index.html, styles.css, script.js)
2. Open `index.html` in your web browser
3. Start managing your court bookings!

### Option 2: Free Online Hosting

#### Using GitHub Pages (Recommended - Completely Free)
1. Create a GitHub account at https://github.com
2. Create a new repository named `bookmaster`
3. Upload these three files to the repository:
   - `index.html`
   - `styles.css` 
   - `script.js`
4. Go to repository Settings → Pages
5. Select "Deploy from a branch" → "main" → "/ (root)"
6. Your site will be live at: `https://[your-username].github.io/bookmaster`

#### Using Netlify (Free Tier)
1. Go to https://netlify.com and create a free account
2. Drag and drop the three files onto the Netlify dashboard
3. Your site will be instantly deployed with a random URL
4. You can customize the URL later if needed

#### Using Vercel (Free Tier)
1. Go to https://vercel.com and create a free account
2. Connect your GitHub account or upload files directly
3. Import the project and deploy
4. Get a free `vercel.app` domain

### Option 3: Using the Files Directly
Since this is a static HTML/CSS/JavaScript application, you can:
- Host it on any web server
- Put it in Dropbox public folder
- Use any free static hosting service

## Usage
1. **Create Booking**: Click "New Booking" and fill in details
2. **Edit Booking**: Click the edit icon on any booking
3. **Delete Booking**: Click the delete icon on any booking
4. **Export to WhatsApp**: Click "Download for WhatsApp" to get formatted text
